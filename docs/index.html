Novo projeto

Para iniciar um novo projeto abra um terminal e faça.

$ mkdir nome-projeto
$ cd nome-projeto
$ mkdoc init

Saída

initializing project...
done.

Editar documento

Para editar seu documento abra um terminal, entre no diretório do seu projeto e faça.

$ mkdir edit

Saída

editing project ...
open with [gedit]:
Enter para abrir com gedit, ou digite seu editor de texto.

Editar template

Para editar seu documento abra um terminal, entre no diretório do seu projeto e faça.

$ mkdir edit -t

Saída

editing project ...
open with [gedit]:
Enter para abrir com gedit, ou digite seu editor de texto.

Visualizar documento

Para visualizar seu documento abra um terminal, entre no diretório do seu projeto e faça.

$ mkdir view

Saída

Seu documento será compilado e seu documento pdf será aberto no evince.

Ênfase

Para enfatizar com negrito ou itálico.

*itálico* ou _itálico_
**negrito** ou __negrito__

também é possível

***itálico e negrito***
___itálico e negrito___

~~~

ou em caso de código inline

um simples `printf("hello world.");` em C.

Links

Para criar um link.

Clique [aqui](http://ubuntu.com/download) para baixar o Ubuntu.

ou

Clique [aqui][ubuntu] para baixar o Ubuntu.
Enter
[ubuntu]: http://ubuntu.com/download

dessa forma um id chamado ubuntu é referenciado logo em baixo, podendo ser usado em outras referências.

Escape Markdown

Para gerar caracteres literais e diferenciá-los da linguagem markdown use \ (backslash).

\* asteriscos literais \*

o mesmo vale para:

  • \ backslash
  • ` backtick
  • * asterisk
  • _ underscore
  • {} curly braces
  • [] square brackets
  • () parentheses
  • # hash mark
  • + plus sign
  • - minus sign (hyphen)
  • . dot
  • ! exclamation mark
